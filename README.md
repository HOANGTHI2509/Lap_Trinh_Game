# Lập Trình Game - Dũng Sĩ Bối Cảnh Hồ Hoàn Kiếm

Một tựa game nhập vai 2D (RPG) được phát triển bằng **Godot Engine 4** và ngôn ngữ **C#**.

## Tính Năng Chính
- **Hệ thống Combat:** Cơ chế tính toán khoảng cách vật lý chuẩn, Dũng sĩ có thể vung kiếm chém quái.
- **AI Quái Vật:** Quái xương tự đi tuần tra (Wander), phát hiện mục tiêu và truy đuổi để cắn người chơi.
- **Hệ thống Cày Cấp (Leveling):** Giết quái để lên cấp, tự động tăng máu và sát thương.
- **Cơ chế Hồi Sinh:** Quái vật chết sẽ tự động tàng hình và chờ 5 giây để hồi sinh ở một vị trí ngẫu nhiên trên bản đồ trống.
- **Giao diện Start/Game Over:** Menu khởi động và kết thúc màn hình với hệ thống nút bấm tàng hình.

## Hướng dẫn cài đặt
1. Yêu cầu có [Godot Engine 4 (.NET version)](https://godotengine.org/)
2. Mở Godot, chọn Import và trỏ vào thư mục project.
3. Nhấn F5 để chơi.

*Dự án trong quá trình hoàn thiện.*
